# 调用速查 · Quick Triggers

> 装完之后，跟你的 agent 说这些话就会触发对应的 skill。

---

## 🌱 twb-daily · 每日浸泡

最常用。每天打开 agent 时说：

```
今天的道
```

或：

```
今日浸泡
```

之后回应今日提问：

```
我想到了：[一句话]
```

或随时记一笔联想：

```
记一笔：[联想内容]
```

回顾过往：

```
翻翻过去 道4
```

或：

```
我对道4 都写过什么
```

用道做第一性原理分析：

```
用道想想 [你想分析的问题]
```

围绕指定道对话：

```
道4 让我想到 [你的联想]
```

---

## 📖 twb-extract · 拆解一本书

```
拆解 /path/to/your-book.md
```

或：

```
提取 ~/Downloads/同读.pdf
```

或：

```
帮我把这本书拆成道法术器势：[书的文件路径]
```

---

## 🔄 twb-structure · 跨书结构整理

审视当前状态（免费、不调 LLM）：

```
审视知识库
```

把刚拆完的新书并入：

```
把 The_Social_Animal 这本书并进来
```

或：

```
同步道库
```

评审候选道：

```
看看候选道
```

或：

```
道索引该不该加新道？
```

全量重新编译 Wiki（费时、费 API 钱，谨慎用）：

```
重新编译知识库
```

单书健康度审视：

```
审视书 Same_as_Ever
```

---

## 完全跳过 extract 和 structure 的最小路径

如果你**已经有了别人分享的道库**，最快路径是：

```bash
# 1. 把别人分享的 道.md 放到你的知识库位置
mkdir -p ~/talkwithbook/dao
cp <shared-dao-file>.md ~/talkwithbook/dao/道.md
```

```
# 2. 在 agent 里
今天的道
```

完成。**整个产品最小可用形态只需要三个文件：一个 SKILL.md + 一个道.md + 一个 agent**。

---

## 复制即用的"上手脚本"

第一次用，建议把这段话整段复制给 agent：

```
我刚装了读后无书（talk without book）。请按这个顺序教我用：

1. 帮我跑一次"审视知识库"看当前状态
2. 如果道库为空，帮我创建一个空的 ~/talkwithbook/dao/道.md
3. 之后告诉我具体的下一步——是拆一本书、还是导入别人的道库
```

agent 会根据你当前的状态给出对应的建议路径。
